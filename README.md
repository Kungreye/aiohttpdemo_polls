# aiohttpdemo_polls
Learn aiohttp by developing a polls application, which is similar to the one in Django tutorial.

Requirements: 
<ul>
    <li>aiohttp</li>
    <li>aiohttp_jinja2</li>
    <li>aiopg</li>
</ul>

Note: python --version = 3.6.4
